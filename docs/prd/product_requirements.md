# Product Requirements Document (PRD) - LiteVault

## 1. Introduction
*   **Product Name:** LiteVault
*   **Version:** 0.1
*   **Status:** Draft

## 2. Product Overview
LiteVault is a lightweight storage solution designed for instant recall.

## 3. Goals & Objectives
*   To provide a fast and efficient way to store data.
*   To ensure data is easily retrievable.

## 4. User Stories
*   As a user, I want to save text quickly so that I don't lose my thoughts.
*   As a user, I want to search my saved items so that I can find them later.
