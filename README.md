# LiteVault

Store lightly. Recall instantly.

## Project Structure

```
/LiteVault
├── /docs                 # Project documentation
│   ├── /prd              # Product Requirements Documents (PRD)
│   ├── /architecture     # Architecture design (Flowcharts, Data models, API design)
│   └── /manual           # User manuals or deployment guides
├── /design               # Design resources
│   ├── /assets           # Logos, icons, static assets
│   └── /prototypes       # Interactive prototypes (Axure/Figma exports)
├── /frontend             # Frontend code (e.g., React, Vue, Flutter)
├── /backend              # Backend code (e.g., Node.js, Go, Python)
├── /scripts              # Automation scripts (DB init, deployment)
├── /docker               # Container configuration (Dockerfile, docker-compose)
├── .gitignore            # Git ignore configuration
├── README.md             # Project overview
└── LICENSE               # Open source license
```
